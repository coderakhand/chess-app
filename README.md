# <p align="center"> Chess Web App </p>

<p > It is a Web application where users can play chess online with friends. It provides user to connect via video call while playing the game. </p>


## Tech Stack

- React
- Tailwindcss + Shadcn
- Websocket
- Webrtc
- Prisma + SQL
